# Redberry Bootcamp Project
 
A modern, web application built with Next.js, featuring a comprehensive design system and scalable architecture.

## 🎯 Project Overview

This project is a professional web application developed as part of the Redberry Bootcamp. It demonstrates modern web development practices, responsive design, and clean code architecture with a focus on user experience and maintainability.

## 🛠️ Tech Stack
 
### Frontend
- **Framework**: Next.js 16.2.2
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **HTTP Client**: Axios
- **Font**: Inter (Google Fonts)

### Development Tools
- **Code Linting**: ESLint
- **Build Tool**: Turbopack
